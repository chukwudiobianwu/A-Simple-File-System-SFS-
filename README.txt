To compile the code type in “make” and hit enter.
Next;
For part 1 type in “ ./diskinfo disk.IMA” will print out the data information
For part 2  type in“./disklist disk.IMA” will print out contents of all the directories
For part 3 type in “ ./diskget disk.IMA foo.txt” will copy the foo.txt file in the disk to the current directory
For part 4 type in “./diskput disk.IMA /subdir1/subdir2/foo.txt” will copy the foo.txt file stored in the current directory to the file system with specified directory 'root/subdir1/subdir2', if the directory exists. And also the file to which would be copied.
